# Frontend Mentor - Interactive rating component

This is a solution to the [Interactive rating component challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/interactive-rating-component-koxS0uSskg).

## Overview

### The challenge
Users should be able to:
- Select and submit a number rating.
- See the "Thank you" card state after submitting.
- View an optimal layout for various device screen sizes.
## My Process

### Built with
- HTML5 & CSS Tailwind
- Mobile-first workflow
- Vanilla JavaScript

### What I learned
I practiced DOM manipulation to capture user input and toggle UI states.
